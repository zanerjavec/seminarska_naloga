HTML

Kaj je html?

  • HTML pomeni Hyper Text Markup Language.
  • HTML opisuje strukturo spletne strani.
  • Elementi HTML povedo brskalniku, kako prikazati vsebino.
  • Elementi HTML so predstavljeni z značkami
  • Značke HTML označujejo dele vsebine, kot so "naslov", "odstavek", "tabela" itd.
  • Brskalniki ne prikazujejo značk HTML, ampak jih uporabljajo za upodabljanje vsebine strani.

Preprost dokument v HTML-ju

Pojasnitev:

  • <!DOCTYPE html>

    pove, da je ta dokument v HTML5.

  • <html lang="en">

    je korenski element strani HTML, atribut lang pa jezik na starni.

  • <head>

    vsebuje meta podatke o elementu.

  • <meta charset="UTF-8">

    značka meta ima več funkcij z atributom charset pa omogoča še uporabo šumnikov.

  • <title>

    element določa naslov dokumenta.

  • <body>

    prikaže se na spletni strani v prevedeni obliki.

  • <h1>

    veliki naslov

  • <p>

    odstavek
