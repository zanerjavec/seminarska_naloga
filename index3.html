Kam pišemo kodo html?

HTML kodo pišemo lahko kar v Notepad(PC) ali TextEdit(MAC). Ni potrebno imeti nekega profesionalnega urejevalnika za to.

Urejevalniki:

  • Notepad
  • Notepad++
  • Brackets
  • Atom
  • MS Visual Studio

1. korak: Odprite Notepad(PC)

Windows 8 ali novejši:

Odprite začetni zaslon (simbol okna spodaj levo na zaslonu). Vnesite beležka.

Windows 7 ali starejši:

Odprite Start > Programi > Pripomočki > Beležnica

1. korak: Odprite TextEdit (Mac)

Odprite Finder> Aplikacije> TextEdit

Spremenite tudi nekatere nastavitve, da boste lahko program pravilno shranili datoteke. V nastavitvah> Oblika> izberite "Navadno besedilo"

Nato pod "Odpri in shrani" potrdite polje z napisom "Prikažite datoteke HTML kot HTML koda namesto oblikovanega besedila".

Nato odprite nov dokument, v katerega boste postavili kodo.

2. korak: Napišite nekaj HTML-ja

Napišite nekaj HTML-ja v Beležnico.

3. korak: Shranite stran HTML

Shranite datoteko v računalniku. Izberite Datoteka> Shrani kot v meniju Beležnica.

Poimenite datoteko "index.html" in nastavite kodiranje na UTF-8 (kar je najprimernejše kodiranje za datoteke HTML).

4. korak: Oglejte si stran HTML v svojem brskalniku

Odprite shranjeno datoteko HTML v svojem najljubšem brskalniku (dvokliknite datoteko ali z desno miškino tipko - in izberite "Za odpiranje uporabi").
